🖥️ Setup Backend (Required)

The backend server runs player stats API and other features.

1 Download & Extract - Same ZIP from above
2 Install Python - Get from python.org (3.8+)
3 Open Terminal/CMD - Navigate to backend folder
4 Install Dependencies
pip install -r requirements.txt
5 Copy Environment File
cp .env.example .env
6 Run Backend Server
python main.py
✓ Backend Running! Should see:
Application startup complete
Uvicorn running on http://0.0.0.0:8000

🔑 Setup API Key (Optional - Demo Mode Works Without)

Add your FortniteAPI.io key for live player stats. Demo mode works perfectly without this!

1 Get Free API Key - Visit fortniteapi.io
2 Sign Up / Login - Create free account
3 Copy API Key - Go to API page, copy your key
4 Edit backend/.env - Paste your key
5 Restart Backend - Stop (Ctrl+C) and run python main.py again
⚠️ Demo Mode: Extension works fine in demo mode with fake players. No API key needed!

▶️ Launch & Test

  1. Make sure backend is running: python main.py
  2. Go to Xbox Cloud Gaming or GeForce NOW
  3. Launch Fortnite
  4. Overlay appears in top-left corner automatically
  5. Press Ctrl+Shift+O to toggle overlay

❓ FAQ

Do I need an API key? ▼

No! Demo Mode works perfectly without a key. You'll see fake player data. If you want real live stats, add your free FortniteAPI.io key.

What's the URL for the extension? ▼

This page: https://zenit45676.github.io/fortnite-cloud-overlay

Repository: https://github.com/ZENIT45676/fortnite-cloud-overlay

Backend: http://localhost:8000 (runs on your computer)

Does overlay slow down my game? ▼

No. Overlay uses <2% CPU and has zero impact on FPS. Cloud gaming streams run fine.

Can I use this on mobile? ▼

Not yet. This is Chrome/Edge desktop extension only. Mobile support coming in future versions.

Is my API key safe? ▼

100% safe. API key stays ONLY in backend/.env on your computer. Extension never sees it. Never transmitted.

Overlay not showing? ▼

Try:

  • Make sure Fortnite is running on cloud gaming
  • Press Ctrl+Shift+O to toggle
  • Check extension in chrome://extensions/
  • Refresh the cloud gaming page
Backend not starting? ▼

Check:

  • Python 3.8+ installed: python --version
  • Dependencies: pip install -r requirements.txt
  • Port 8000 not in use
  • Run from backend folder
